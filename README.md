# 📚 Student Attendance Register

A lightweight and responsive **Student Attendance Register** built using **HTML5, CSS3, and Vanilla JavaScript**. The application allows users to manage student attendance efficiently through a simple and intuitive interface while storing data locally using the browser's Local Storage.

---

## 🚀 Live Demo

🔗 **Website:**  
https://nqobile-21.github.io/student-attendance-register/ 
 
---

## 📖 Project Overview

This application was developed as a coding challenge using **Vanilla JavaScript**. It enables users to manage student attendance without requiring a database or backend server.

The application stores student records in the browser using **Local Storage**, allowing data to persist after refreshing the page.

---

## ✨ Features

- ✅ Add new students
- ✅ Edit student details
- ✅ Delete students
- ✅ Search students by ID or name
- ✅ Filter students by attendance status
- ✅ Mark students as Present or Absent
- ✅ Automatic alphabetical sorting
- ✅ Student number validation
- ✅ Student name validation
- ✅ Attendance summary dashboard
- ✅ Current date display
- ✅ Local Storage support
- ✅ Responsive design
- ✅ Dark Mode

## 📋 Student Number Format

Student numbers must follow the format below:

```
PT.2025.ABC123
```

Examples:

```
PT.2022.N8R9K8

```



## 📊 Dashboard

The attendance dashboard automatically displays:

- Total Students
- Present Students
- Absent Students
- Attendance Percentage

The summary updates automatically whenever attendance records change.

---

## 💾 Data Storage

The application uses **Local Storage** to save student information.

This means:

- Student records are automatically saved.
- Data remains available after refreshing the browser.
- No database installation is required.
